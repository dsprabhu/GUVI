1) Clone the give repo.
git clone https://github.com/Vennilavan12/Brain-Tasks-App.git

2) Build the React App.
In your project root:
$ cd Brain-Tasks-App
$ sudo npm run build
This generates a build/ folder with static HTML, CSS, and JS files.
But we have already build directory it has all the files, so no need to run (npm run build).

3)Create a Dockerfile.

FROM nginx:alpine
WORKDIR /usr/share/nginx/html
COPY dist/ .
EXPOSE 3000
CMD ["nginx", "-g", "daemon off;"]

4)Build the image.
docker build -t brain-tasks-app:1.0 .

5)Push Image to Docker Registry.
docker login (login docker hub registry)
docker tag brain-tasks-app:1.0 your-dockerhub-username/brain-tasks-app:1.0
docker push my-dockerhub-username/brain-tasks-app:1.0


6)Create Kubernetes Deployment (deployment.yaml).

apiVersion: apps/v1
kind: Deployment
metadata:
  name: brain-tasks-deployment
spec:
  replicas: 2
  selector:
    matchLabels:
      app: brain-tasks
  template:
    metadata:
      labels:
        app: brain-tasks
    spec:
      containers:
      - name: brain-tasks
        image: my-dockerhub-username/brain-tasks-app:1.0
        ports:
        - containerPort: 3000


kubectl apply -f deployment.yaml (apply deployment)

7) Expose the Service to connect outside world. (service.yaml)

apiVersion: v1
kind: Service
metadata:
  name: brain-tasks-app-service
spec:
  type: LoadBalancer
  selector:
    app: brain-tasks
  ports:
    - protocol: TCP
      port: 3000
      targetPort: 3000

kubectl apply -f service.yaml (apply service deployment)

8) To access the App

kubectl get svc (to check the service)

Use the external IP (if on cloud) or port-forward locally:

kubectl port-forward svc/brain-tasks-app-service 8080:3000

9)verify http://localhost:8080
